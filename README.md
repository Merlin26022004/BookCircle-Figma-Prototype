# BookCircle – Community Virtual Library

An Agile-based digital solution aligned with SDG 4 (Quality Education).

## About
BookCircle is a community-driven virtual library prototype that allows users to donate, borrow, wishlist, and share books, encouraging inclusive learning and reading culture.

## SDG Focus
- SDG 4 – Quality Education
